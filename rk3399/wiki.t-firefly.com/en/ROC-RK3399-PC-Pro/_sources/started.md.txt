Introduction

ROC-RK3399-PC Pro is a mini PC motherboard, about the size of a small
phone.


Dual-core Cortex-A72 up to 1.8GHz & Quad-core Cortex-A53 up to 1.5GHz
Six Core Processor

Mali-T864 GPU，Support OpenGL ES1.1/2.0/3.0, OpenCL1.2, DirectX11.1.

LPDDR4 RAM(2/4 GB)

1 x USB 2.0(Type-A)

1 x USB 3.0(Type-A)

10/100/1000 Mb eth interface

IR Infrared Module

HDMI 2.0a with HDCP 2.2(4K@60Hz)

1 x Type-C Interfaces -- Type-C0 support DisplayPort 1.2 (4K@60Hz)、
USB3.0

3 x MIPI interfaces -- MIPI0 only for DSI, MIPI1 for DSI or CSI, MIPI2
only for CSI, The Maximum resolution of DSI support 1080P@60fps

eDP 1.3(Maximum resolution support 2K@60fps)

Wifi(2.4G && 5G)

BT 5.0

M.2PCIE-M.key interface

I2S，Support 8 Channels

High speed eMMC Expansion Interface

MicroSD (TF) Card Slot

RTC : battery model -- CR1220 ; Battery seat -- CR1220,BS-12-A3B0J002

60 Pin Expansion -- Interfaces:I2C、SPI、Uart、I2S...

Shipping list(For reference only)
